## 一、问题定义与数据概览

### 1.1 任务

从光电容积脉搏波（PPG）信号中预测收缩压（SBP）和舒张压（DBP），实现无袖带连续血压监测。

**信号规格**：

| 参数           | 值                                          |
| ------------ | ------------------------------------------ |
| 采样率 Fs       | 125 Hz                                     |
| 窗口长度         | 2048 点（≈16.4 s）                            |
| 1D-ResNet 输入 | (Batch, 3, 2048)，3 通道 = PPG + dPPG + d²PPG |
| GPR 输入       | 7 维特征向量                                    |
| 输出           | 2 维（SBP, DBP），单位 mmHg                      |

### 1.2 两套数据集

| 数据集                  | 训练样本        | 验证样本 | 测试样本 | 数据形式                       |
| -------------------- |:-----------:|:----:|:----:| -------------------------- |
| **公开数据集**（源域）        | 4745 PPG 波形 | 1577 | 1582 | .mat v5（scipy 可读）          |
| **自建数据集**（目标域，原始PPG） | 253         | —    | 152  | .mat v7.3 cell 数组（h5py 解析） |
| **自建数据集**（目标域，78维特征） | 774         | —    | 273  | .mat v5                    |

> 自建数据集的原始 PPG（v7.3 cell 数组）在 v1.x 开发阶段尚未就绪，当时仅有 78 维特征可用。v1.x DL 路线因此采用了公开数据集内部 7:3 拆分作为过渡方案。v2.0 final 中原始 PPG 到位后，DL 路线升级为真正的公开→自建跨域迁移。

### 1.3 核心挑战：Domain Shift

公开数据集（源域）上训练好的模型直接用于自建数据集（目标域）时，性能显著退化。Base 基线（不做任何适应）在自建测试集上：SBP MAE = 11.36 mmHg，DBP MAE = 7.18 mmHg——远超 AAMI 临床可接受标准（MAE ≤ 5 mmHg, STD ≤ 8 mmHg）。

差异来源：采集设备（传感器型号/光电特性）、人群分布（年龄/血管弹性）、采集协议（静息 vs 日常佩戴）。

## 二、1D-ResNet 深度学习路线

### 2.1 模型架构

#### 2.1.1 总体结构

模型定义在 `model.py` 中的 `ResNet1D` 类，总参数量约 **500K**。

```
Input (B, 3, 2048)
  │
  ├─ conv1:  Conv1d(in=3, out=64, k=7, stride=2, pad=3, bias=False)
  ├─ bn1:    BatchNorm1d(64)
  ├─ relu
  ├─ maxpool: MaxPool1d(k=3, stride=2, pad=1)
  │   └─→ (B, 64, 512)
  │
  ├─ layer1: 2×BasicBlock(64→64,  stride=1)
  │   └─→ (B, 64, 512)     ← 保持分辨率，提取精细局部特征
  │
  ├─ layer2: 2×BasicBlock(64→128, stride=2)
  │   └─→ (B, 128, 256)     ← 通道翻倍，1×1 投影 shortcut
  │
  ├─ layer3: 2×BasicBlock(128→256, stride=2)
  │   └─→ (B, 256, 128)     ← 通道再翻倍
  │
  ├─ gap:    AdaptiveAvgPool1d(1)
  │   └─→ (B, 256, 1)       ← 全局平均池化，消除空间维度
  │
  ├─ flatten → (B, 256)     ← 256-d 特征向量（CORAL 域适应对齐目标）
  │
  └─ fc:     Linear(256, 2)
      └─→ (B, 2)            ← [SBP, DBP]
```

#### 2.1.2 BasicBlock1D 残差块

```python
class BasicBlock1D(nn.Module):
    def __init__(self, in_channels, out_channels, stride=1):
        # 主路径
        self.conv1 = Conv1d(in_ch, out_ch, k=3, stride=stride, pad=1, bias=False)
        self.bn1   = BatchNorm1d(out_ch)
        self.conv2 = Conv1d(out_ch, out_ch, k=3, stride=1, pad=1, bias=False)
        self.bn2   = BatchNorm1d(out_ch)

        # shortcut: 维度不匹配时用 1×1 Conv 投影
        self.shortcut = Sequential()
        if stride != 1 or in_ch != out_ch:
            self.shortcut = Sequential(
                Conv1d(in_ch, out_ch, k=1, stride=stride, bias=False),
                BatchNorm1d(out_ch),
            )

    def forward(self, x):
        out = ReLU(BN1(Conv1(x)))
        out = BN2(Conv2(out))
        out += self.shortcut(x)   # 残差连接
        out = ReLU(out)
        return out
```

#### 2.1.3 架构设计理由

| 设计选择                 | 理由                                                                             |
| -------------------- | ------------------------------------------------------------------------------ |
| **1D 卷积**            | PPG 是一维时序信号，1D 卷积的局部感受野天然匹配脉搏波形态特征（收缩峰宽度 ≈100–200 ms = 12–25 采样点）；参数量远低于 2D 卷积 |
| **残差连接**             | 恒等映射路径缓解深层网络的梯度消失，使 6-block 深层架构可正常训练                                          |
| **大卷积核首层**（k=7, s=2） | 感受野 56 ms，一次捕获充分的初始波形形态信息                                                      |
| **GAP 替代 Flatten**   | 256-d 池化向量天然正则化（参数量为 0），同时作为后续 CORAL 域适应的对齐目标                                  |
| **Bias=False**       | BatchNorm 已包含可学习偏移参数，卷积层 bias 冗余                                               |

#### 2.1.4 逐层参数表

| 层         | 输出形状          | 参数量            | 备注                      |
| --------- | ------------- | -------------- | ----------------------- |
| Input     | (B, 3, 2048)  | 0              | 3 通道：PPG + dPPG + d²PPG |
| conv1     | (B, 64, 1024) | 3×64×7 = 1,344 | k=7, s=2, pad=3         |
| bn1       | (B, 64, 1024) | 128            | γ 初始=1, β 初始=0          |
| maxpool   | (B, 64, 512)  | 0              | k=3, s=2, pad=1         |
| layer1[0] | (B, 64, 512)  | ≈24,576        | s=1, 无投影                |
| layer1[1] | (B, 64, 512)  | ≈24,576        | s=1, 无投影                |
| layer2[0] | (B, 128, 256) | ≈82,432        | s=2, 1×1 投影(64→128)     |
| layer2[1] | (B, 128, 256) | ≈98,304        | s=1, 无投影                |
| layer3[0] | (B, 256, 128) | ≈328,192       | s=2, 1×1 投影(128→256)    |
| layer3[1] | (B, 256, 128) | ≈393,216       | s=1, 无投影                |
| GAP       | (B, 256, 1)   | 0              | AdaptiveAvgPool1d       |
| FC        | (B, 2)        | 256×2 = 512    |                         |
| **总计**    | —             | **≈500K**      |                         |

| 层类型         | 初始化方法             | 参数                                  |
| ----------- | ----------------- | ----------------------------------- |
| Conv1d      | `kaiming_normal_` | mode='fan_out', nonlinearity='relu' |
| BatchNorm1d | 常数初始化             | weight=1, bias=0                    |
| Linear      | 正态分布              | μ=0, σ=0.01, bias=0                 |

### 2.2 数据预处理

#### 2.2.1 三通道生成

在 `model.py` 中由 `preprocess_ppg()` 实现：

对每条 PPG 信号 $x(t), t \in \{0, \Delta t, 2\Delta t, \dots\}$，其中 $\Delta t = 1/F_s = 0.008\text{s}$：

$$
\begin{aligned}
C_1(t) &= x(t)                    &&\text{原始 PPG} \\
C_2(t) &= \frac{dx}{dt}           &&\text{一阶导数（血流速度变化）} \\
C_3(t) &= \frac{d^2x}{dt^2}       &&\text{二阶导数（加速度/波形拐点）}
\end{aligned}
$$

使用 `numpy.gradient` 中心差分法计算数值导数（`edge_order=1` 边界用一阶差分）。内部以 float64 精度计算，最终转为 float32。

堆叠为 `(N, 3, 2048)` 形状的 numpy 数组。

#### 2.2.2 归一化

**输入归一化**（`normalize_3ch` / `apply_normalize_3ch`）：

三个通道各自独立 z-score 标准化。对通道 $c \in \{0,1,2\}$，在整个训练集的该通道全体采样点（$N \times 2048$ 个值）上计算 $\mu_c$ 和 $\sigma_c$：

$$
C'_c = \frac{C_c - \mu_c}{\sigma_c}
$$

归一化参数 $\{\mu_c, \sigma_c\}_{c=0}^2$ 在基础训练阶段由训练集计算，持久化保存至 `models/norm_params.npz`，后续所有阶段（微调、评估）均使用同一组参数。

**标签归一化**（`normalize_bp`）：

SBP 和 DBP 联合 z-score 标准化——两个目标共享均值和标准差在同一个 2 维向量中。这保留了 SBP/DBP 之间的相对尺度关系：

$$
Y_{\text{norm}} = \frac{Y - \mu_{\text{bp}}}{\sigma_{\text{bp}}}, \quad
Y = [\text{SBP}, \text{DBP}] \in \mathbb{R}^{N \times 2}
$$

#### 2.2.3 自建数据集加载

自建 PPG 存储为 MATLAB v7.3 HDF5 cell 数组（`dataset/PPG/TrainPPG_cell.mat`）。每个 cell 为 2048×1 的信号，以 HDF5 对象引用（object reference）形式逐点存储——每个采样点一个独立引用。

`load_self_dataset()` 函数（`model.py`）通过 h5py 逐列遍历引用数组，逐引用解引用（dereference）提取浮点数值，重构为 `(n_samples, 2048)` 的数值矩阵。该过程是数据加载链路中技术复杂度和 I/O 开销最高的环节（每信号 2048 次 h5py dereference 操作）。

自建 BP 标签（`dataset/BP/TrainSBP.mat` 等）同样为 v7.3 HDF5 格式，使用 `load_v73_var()` 函数读取，自动处理 h5py 存储的转置问题（MATLAB `(1,N)` → h5py `(N,1)`）。

### 2.3 基础训练

#### 2.3.1 训练脚本与数据流

**脚本**：`train_base_model.py`（238 行）

**数据流**：

```
公开数据集 TrainPPG.mat (4745×2048)
  → preprocess_ppg()    → (4745, 3, 2048)
  → normalize_3ch()     → 归一化 + 保存 params
  → normalize_bp()      → 标签归一化

验证集（1577 样本）仅用于监控，不参与训练。`val_freq=50` 即每 50 次权重更新在全量验证集上计算一次验证损失。

| `LR_GAMMA`      | 0.5  | 学习率衰减因子           |
| `WEIGHT_DECAY`  | 1e-4 | Adam 权重衰减（L2 正则化） |
| `GRAD_CLIP`     | 5.0  | 梯度裁剪最大范数          |
| `VAL_FREQ`      | 50   | 验证间隔（迭代次数）        |

#### 2.3.3 训练循环

```
初始化: model, criterion=MSE, optimizer=Adam, scheduler=StepLR

for epoch = 1 to 80:
    for batch in train_loader:
        y_pred = model(x_batch)
        loss = MSE(y_pred, y_batch)
        optimizer.zero_grad()
        loss.backward()
        clip_grad_norm_(max_norm=5.0)
        optimizer.step()

        every 50 iters:
            val_loss = MSE(model(X_val), Y_val)
            if val_loss < best_val_loss:
                save model → base_model_best.pth

    scheduler.step()  # 每 25 epoch 学习率 ×0.5
    epoch_end_val_loss = MSE(model(X_val), Y_val)
    if improved: save model → base_model_best.pth

save final → base_model_final.pth
```

### 2.4 跨域微调

#### 2.4.1 微调脚本与数据流

**脚本**：`fine_tune_model.py`（≈280 行）

**数据流**：

```

#### 2.4.3 三种机制详述

**① 参数选择性冻结**

遍历模型的所有 `named_parameters()`，名称以 `conv1.`、`bn1.`、`layer1.`、`layer2.` 开头的参数设置 `requires_grad=False`。冻结后仅 `layer3` 和 `fc` 参与梯度更新。

设计依据：

- **conv1**（7×1 大卷积核）：学习最底层的局部波形基元（上升支、波峰、重搏波切迹），跨域高度通用；
- **layer1 & layer2**：提取中层的形态组合特征，跨域共享属性强；
- **layer3**（最深层）和 **fc**（输出头）：编码与具体血压值直接关联的高层语义表示，需要在新域重新校准。

冻结后有效可训练参数约为原来的 **60%**，显著降低小样本过拟合风险。

**② 混合批采样**

每训练批构成为：16 条源域样本 + 16 条目标域样本（`FT_DATA_RATIO=0.5`）。

每 epoch 开始时，分别对源域索引（4745 条）和目标域索引（253 条）进行独立随机重排（`torch.randperm`），然后按 `batch_pub` 和 `batch_self` 的采样数从各自重排序列中依次截取子批次。

仅用目标域数据微调 → 灾难性遗忘（参数沿梯度方向完全偏离源域学到的解空间）；仅用源域数据 → 无法适应新域。混合批采样在每一批中同时注入两个域的梯度信号，在源域知识保持和目标域适应之间求最优权衡。

由于目标域样本数（253）远小于源域（4745），目标域样本在每个 epoch 内会被重复使用更多次（约 4745×0.5 / 253×0.5 ≈ 18.8 倍）。

**③ 知识蒸馏正则化**

教师模型 = 冻结的基础模型（eval 模式，全部参数 `requires_grad=False`）。

总损失函数（蒸馏模式，`USE_DISTILL=True`）：

$$
\mathcal{L}_{\text{total}} = \mathcal{L}_{\text{MSE}}(\hat{y}_{\text{tgt}}, y_{\text{tgt}})
+ (1-\alpha) \cdot \mathcal{L}_{\text{MSE}}(\hat{y}_{\text{src}}, y_{\text{src}})
+ \alpha \cdot \mathcal{L}_{\text{MSE}}\left(\frac{\hat{y}_{\text{src}}}{T}, \frac{y_{\text{teacher}}}{T}\right) \cdot T^2
$$

其中：

- 第一项：目标域数据的 MSE 任务损失
- 第二项：源域数据的 MSE 任务损失（保持基础任务能力）
- 第三项：蒸馏损失——学生源域预测与教师源域预测之间的 MSE（温度缩放后）
- $\alpha = 0.4$：蒸馏权重
- $T = 3.0$：温度系数（T > 1 软化输出分布，携带丰富的"暗知识"）
- 乘以 $T^2$：恢复温度缩放导致的梯度幅值衰减

消融模式下（`USE_DISTILL=False`），第三项被跳过，总损失 = 第一项 + 第二项（两者权重均为 1.0），教师模型不被加载。

### 2.5 双数据集评估

#### 2.5.1 评估脚本

**脚本**：`evaluate_model.py`（311 行）

**评估模式**：`EVAL_MODE = 'base' | 'finetuned' | 'both'`

#### 2.5.2 评估流程

对每个已加载的模型（base / finetuned），分别执行：

1. **公开测试集评估**（源域内，1582 样本）：衡量模型在其训练分布上的性能保持
2. **自建测试集评估**（跨域核心指标，152 样本）：衡量模型在目标域上的泛化能力

每个测试集上计算四项指标（单位：mmHg）：

| 指标               | 公式                                                                                                                    | 含义                |
| ---------------- | --------------------------------------------------------------------------------------------------------------------- | ----------------- |
| **MAE**          | $\frac{1}{n}\sum_{i=1}^{n}\|\hat{y}_i - y_i\|$                                                                        | 平均绝对误差，预测准确度      |
| **STD**          | $\text{std}(\hat{y}_i - y_i)$                                                                                         | 预测误差标准差，预测稳定性     |
| **Pearson r**    | $\frac{\sum(\hat{y}_i-\bar{\hat{y}})(y_i-\bar{y})}{\sqrt{\sum(\hat{y}_i-\bar{\hat{y}})^2 \cdot \sum(y_i-\bar{y})^2}}$ | 预测值与真值的线性相关强度     |
| **Bland-Altman** | Bias $\pm$ 1.96·STD                                                                                                   | 95% 一致性界限，临床方法论标准 |

每模型 × 每测试集生成 3 张 150 DPI PNG：

### 2.6 关键改进：从内部拆分到真实跨域

#### 2.6.1 改进前（v1.x 内部拆分方案）

v1.x DL 路线采用公开数据集内部 7:3 拆分：

## 三、GPR 高斯过程回归路线

### 3.1 特征体系与 78→7 降维

#### 3.1.1 78 维特征来源

GPR 路线的输入特征来自 v1.0 MATLAB 特征提取流水线（`find_parameter_amend.m`, 861 行）。该流水线对每条 PPG 信号经过四级预处理后，提取覆盖 5 个特征域的 78 维特征：

| 特征域    | 子函数          | 数量  | 代表特征                                                                      |
| ------ | ------------ |:---:| ------------------------------------------------------------------------- |
| 时域     | `find_time`  | 22  | 收缩时间(ST)、舒张时间(DT)、波幅(p_t)、脉宽(width1/2)、波形因子(K)、每搏量(SV)、外周阻力(TPR)、上升支斜率(k) |
| 面积     | `find_area`  | 5   | 收缩期面积(S_S)、舒张期面积(S_D)、面积比(AreaR)、面积占比(SR_SA, SR_DA)                       |
| 一阶导数   | `find_diff1` | 9   | 导数峰值(pks_d1)、变化速率、加速度指标                                                   |
| 频域     | `find_freq`  | 26  | FFT 基频 f1/谐波 f2（三次样条插值精化）、15 子带能量(0.25-20 Hz)及能量比                         |
| 小波+HHT | `find_WT`    | 17  | db6 DWT 细节能量(Ed1-4)、EEMD-IMF 能量矩(E1-9)、HHT 边际谱子带能量(Eb1-4)                 |

特征数据以 `.mat v5` 格式预提取存储：`TrainParameter.mat` (N×78) 和 `TestParameter.mat` (N×78)。公开数据集和自建数据集均有对应的 Parameter 文件。

#### 3.1.2 78→7 特征选择

GPR 的计算复杂度与特征维度强相关。精确 GP 的核矩阵计算为 O(n²d)（d 为特征维度），存储为 O(n²)。78 维特征下，4745 样本的核矩阵为 4745² ≈ 22.5M 元素，内存占用 ≈180 MB。如果使用 ARD 核（每特征一个长度尺度参数），超参数数量为 79 个，优化收敛极慢。

**因此选择从 78 维特征中筛选最有效的 7 维子集**。

#### 3.1.3 选取的 7 个特征

特征子集沿用 v1.0 ε-SVR 方案（`train_SBP.m` / `train_DBP.m`）中的 `dataSortList`：

```matlab
dataSortList = [26, 7, 32, 43, 44, 23, 42];
```

| 编号  | 名称         | 特征域 | 生理含义                   |
|:---:| ---------- |:---:| ---------------------- |
| 7   | **width1** | 时域  | 2/3 峰值高度处的脉宽，反映动脉管壁弹性  |
| 23  | **SV**     | 时域  | 每搏输出量生理估算，与脉压直接相关      |
| 26  | **f2**     | 频域  | FFT 二次谐波频率，反映心率变异性     |
| 32  | **fs4**    | 频域  | 能量比指标，表征频域能量分布形态       |
| 42  | **Z**      | 时域  | 每搏输出量指数 Z = p_t/T×s    |
| 43  | **SR_DA**  | 面积  | 舒张面积占比 = S_D/(S_S+S_D) |
| 44  | **SR_SA**  | 面积  | 收缩面积占比 = S_S/(S_S+S_D) |

这 7 个特征覆盖了 3 个特征域（时域 3 + 面积 2 + 频域 2），保留了从不同角度描述 PPG 形态的互补信息。

#### 3.1.4 计算量分析

| 指标                   | 78 特征         | 7 特征          | 降幅         |
| -------------------- |:-------------:|:-------------:|:----------:|
| 核矩阵尺寸                | 4745² ≈ 22.5M | 4745² ≈ 22.5M | 相同（仅依赖 n）  |
| 每维协方差计算              | O(78²) ≈ 6084 | O(7²) ≈ 49    | **↓99.2%** |
| ARD 超参数数             | 79            | 8             | ↓89.9%     |
| GP 训练时间（估算）          | 分钟级           | 秒级            | ↓90%+      |
| `fitrgp` 'exact' 可行性 | 可能但慢          | 快速            | —          |

核矩阵大小不随特征维度变化（仅依赖于样本数 n），但 GP 超参数优化（如核函数的长度尺度参数）的每次迭代需要对特征空间做操作。使用单长度尺度的 Matérn 5/2 核（仅 2 参数：长度尺度 σ_l + 信号方差 σ_f²），超参数优化在 7 特征下非常快速。

### 3.2 Matérn 5/2 核函数

Matérn 核函数族的形式为：

$$
k(r) = \sigma_f^2 \cdot \frac{2^{1-\nu}}{\Gamma(\nu)} \left(\sqrt{2\nu}\frac{r}{\ell}\right)^\nu K_\nu\left(\sqrt{2\nu}\frac{r}{\ell}\right)
$$

其中 $r = \|x - x'\|_2$，$\nu$ 为平滑度参数，$\ell$ 为长度尺度，$\sigma_f^2$ 为信号方差，$K_\nu$ 为修正贝塞尔函数。

当 $\nu = 5/2$ 时，形式简化为：

$$
k_{5/2}(r) = \sigma_f^2 \cdot \left(1 + \frac{\sqrt{5}r}{\ell} + \frac{5r^2}{3\ell^2}\right) \exp\left(-\frac{\sqrt{5}r}{\ell}\right)
$$

**选择 Matérn 5/2 的理由**：

- $\nu = 5/2$ 对应**二阶均方可微**的平滑先验，即样本路径具有连续的一阶和二阶导数；
- RBF 核（$\nu \to \infty$）：无限阶可微，可能**过度平滑**，不适合具有中等粗糙度的生理信号映射；
- Matérn 3/2 核（$\nu = 3/2$）：仅一阶可微，可能**欠平滑**；
- Matérn 5/2 在两个极端之间取得平衡，更适合建模 PPG→BP 这种具有中等平滑度的生理映射关系。

**MATLAB 实现**：

```matlab
gprSBP = fitrgp(X_train_norm, Y_sbp, ...
    'KernelFunction', 'matern52', ...
    'Standardize', false, ...    % 已在外部手动 z-score 标准化
    'FitMethod', 'exact', ...    % 基础训练：精确 GP
    'PredictMethod', 'exact');
```

### 3.3 两阶段训练

#### 3.3.1 阶段一：基础模型训练（`train_gpr.m`, 120 行）

**数据流**：

```
公开 TrainParameter.mat (4745×78)
  → 自动检测方向 (N×78 或 78×N) + 转置
  → X_train = X_full(:, [7,23,26,32,42,43,44])   ← 78→7
  → 剔除 Inf/NaN 行
  → z-score 标准化 (基于训练集统计)
  → fitrgp (Kernel=matern52, FitMethod=exact) ×2 (SBP, DBP)
  → 训练集自评估 (MAE/STD)
  → 保存 gpr_base.mat
```

**归一化**：z-score 标准化（非 mapminmax 的 [0,1] 缩放），基于训练集全体样本的每特征均值和标准差。标准差为 0 的特征被替换为 1（避免除零）。

**输出**：`gpr_models/gpr_base.mat`

| 变量          | 说明                      |
| ----------- | ----------------------- |
| `gprSBP`    | 训练好的 SBP GP 回归模型对象      |
| `gprDBP`    | 训练好的 DBP GP 回归模型对象      |
| `X_mean`    | 7 维特征均值向量               |
| `X_std`     | 7 维特征标准差向量              |
| `KERNEL`    | 核函数名称字符串 (`'matern52'`) |
| `Parameter` | 7 个特征名称的 cell 数组        |

#### 3.3.2 阶段二：跨域微调（`finetune_gpr.m`, 123 行）

**数据流**：

```
公开 TrainParameter.mat (4745×78) → 78→7 → X_pub (4745×7)
自建 TrainParameter.mat  (774×78) → 78→7 → X_self (774×7)
  → 纵向拼接 X_all = [X_pub; X_self] (5519×7)
  → 联合 z-score 标准化 (基于拼接后全体样本)
  → fitrgp (Kernel=matern52, FitMethod=sd) ×2  ← SD 子集近似
  → 训练集自评估
  → 保存 gpr_finetuned.mat
```

**拟合方法变更**：

| 阶段   | 样本数  | 拟合方法    | 复杂度               | 原因                         |
| ---- |:----:| ------- | ----------------- | -------------------------- |
| 基础训练 | 4745 | `exact` | O(n³) ≈ 1.07×10¹¹ | 4745 样本在 exact GP 下可接受     |
| 跨域微调 | 5519 | `sd`    | O(nm²)            | 样本量增大，exact 的 O(n³) 代价显著增加 |

SD（Subset of Data）方法选择 $m \ll n$ 个诱导点（inducing points）来近似完整的 n×n 核矩阵，将计算复杂度从 O(n³) 降至 O(nm²)。MATLAB `fitrgp` 的 `'sd'` 方法默认 $m = \min(2000, \lfloor n/2 \rfloor)$。

### 3.4 2×2 交叉评估

**脚本**：`evaluate_gpr.m`（224 行，含 54 行内嵌画图函数）

### 3.5 预测不确定性量化

GPR 的核心优势：每个预测值自带不确定性估计。MATLAB `fitrgp` 的 `predict` 方法返回三个值：

```matlab
[ypred, ysd, yci] = predict(gprModel, X_test);
```

| 输出      | 含义                                                      |
| ------- | ------------------------------------------------------- |
| `ypred` | 后验预测均值 $\mathbb{E}[y_* \mid X, y, X_*]$                 |
| `ysd`   | 后验预测标准差 $\sqrt{\mathbb{V}[y_* \mid X, y, X_*]}$         |
| `yci`   | 95% 置信区间 $[\hat{y} - 1.96\sigma, \hat{y} + 1.96\sigma]$ |

在 Line 图中，GPR 版叠加 95% CI 阴影带（红色半透明填充），直观展示模型对不同样本的预测置信度。在临床决策场景中：置信区间较窄的预测 → 可较信赖；置信区间较宽的预测 → 应审慎对待或建议用袖带血压计确认。

## 四、方案对比与互补关系

### 4.1 1D-ResNet vs GPR

| 维度        | 1D-ResNet DL              | GPR                                |
| --------- | ------------------------- | ---------------------------------- |
| **输入**    | 原始 PPG 波形 (3×2048)        | 7 维手工特征向量                          |
| **特征来源**  | 端到端自动学习                   | v1.0 MATLAB 特征工程预提取                |
| **模型参数量** | ≈500K                     | 非参数（核方法，取决于训练样本数）                  |
| **训练范式**  | 小批随机梯度下降                  | 核矩阵构建 + 超参数边际似然优化                  |
| **跨域策略**  | 冻结+混合批+知识蒸馏               | 源域+目标域数据拼接联合训练                     |
| **不确定性**  | 点预测（无原生不确定性）              | 后验预测分布（均值 + 方差 + 95% CI）           |
| **计算瓶颈**  | GPU 前向/反向传播               | 核矩阵 O(n³) / O(nm²)                 |
| **可解释性**  | 低（黑盒端到端）                  | 中（特征有明确生理含义）                       |
| **标注需求**  | 全量目标域 (253)               | 全量目标域 (774 特征样本)                   |
| **语言/平台** | Python 3.9+, PyTorch 2.0+ | MATLAB R2019b+, Statistics Toolbox |

### 4.2 两路线的互补场景

| 场景                    | 推荐路线                           | 理由             |
| --------------------- | ------------------------------ | -------------- |
| 目标域有原始 PPG + 全量 BP 标注 | **1D-ResNet + 微调**             | 端到端优化，自动发现最优特征 |
| 目标域仅有手工特征 + 需要置信区间    | **GPR**                        | 特征级迁移，原生不确定性量化 |
| 目标域标注极少 (32~64 条)     | **1D-ResNet + DA (CORAL/LwF)** | 小样本域适应         |
| 医疗合规要求可解释性            | **GPR**                        | 7 个特征均有明确的生理含义 |

### 5.1 实验设定

| 参数 | 值 |

| 源域验证 | 公开数据集 Val 1577 样本 |
| 源域测试 | 公开数据集 Test 1582 样本 |
| 目标域训练 (DL) | 自建数据集 Train 253 样本（原始 PPG 波形） |
| 目标域测试 (DL) | 自建数据集 Test 152 样本 |
| 目标域训练 (GPR) | 自建数据集 Train 774 样本（78→7 特征） |
| 目标域测试 (GPR) | 自建数据集 Test 273 样本 |
| 评估指标 | MAE↓, STD↓（单位 mmHg）+ Pearson r + Bland-Altman |
| 临床标准 | AAMI: MAE ≤ 5 mmHg, STD ≤ 8 mmHg |
| 模型检查点 | `base_model_best.pth`, `finetuned_model_best.pth` |

### 5.2 1D-ResNet 跨域微调结果

基础训练：公开全量 4745 样本。微调：自建 253 样本，冻结浅层 + 混合批采样 + **知识蒸馏**（`USE_DISTILL=True`, α=0.4, T=3.0）。

评估框架：2 测试集 × 2 模型 = 4 组。

| 模型 | 测试集 | SBP MAE↓ | SBP STD↓ | DBP MAE↓ | DBP STD↓ |
|------|--------|:---:|:---:|:---:|:---:|
| base | 公开 (1582) | 2.42 | 3.61 | 1.30 | 2.03 |
| base | 自建 (152) | 12.87 | 11.61 | 3.85 | 4.13 |
| finetuned | 公开 (1582) | 10.84 | 12.24 | 5.56 | 6.77 |
| **finetuned** | **自建 (152)** | **6.92** | **7.81** | **2.27** | **2.53** |

**分析**：

- **源域内性能**（base → 公开测试）：SBP MAE=2.42, DBP MAE=1.30，基础模型在源域上表现优秀，满足 AAMI 标准。
- **直接跨域性能**（base → 自建测试）：SBP MAE=12.87, DBP MAE=3.85。SBP 远超 AAMI 标准，证实 Domain Shift 的严重性。DBP（MAE=3.85）的跨域退化远小于 SBP（12.87），说明 DBP 映射在两个域之间有更强的结构性共性。
- **蒸馏微调后跨域性能**（finetuned → 自建测试）：**SBP 从 12.87 → 6.92（↓46.2%），DBP 从 3.85 → 2.27（↓41.0%）**。DBP MAE=2.27 已满足 AAMI MAE≤5 标准。
- **蒸馏 vs 无蒸馏消融对比**：

| 配置 | SBP MAE (自建) | SBP STD | DBP MAE | DBP STD |
|------|:---:|:---:|:---:|:---:|
| 无蒸馏（`USE_DISTILL=False`） | 7.57 | 7.67 | 2.08 | 2.50 |
| **有蒸馏**（`USE_DISTILL=True`） | **6.92** | 7.81 | 2.27 | 2.53 |
| 改善 | ↓0.65 ✅ | — | ↑0.19 | — |

蒸馏在 SBP 上带来 0.65 mmHg 的改善（8.6% 相对降低），DBP 基本持平。教师模型的软标签约束对 SBP 这种个体差异大、迁移难度高的目标更为有效。
- **源域性能保留**（finetuned → 公开测试）：SBP MAE 从 2.42→10.84，出现明显退化。这反映微调过程中混合批与蒸馏无法完全阻止灾难性遗忘——模型在适应目标域的同时，在源域的预测能力有所下降。

### 5.3 GPR 跨域迁移结果（7 特征）

基础训练：公开 4745×7（Matérn 5/2, exact）→ 微调：公开 4745+自建 774=5519×7 拼接（Matérn 5/2, sd 子集近似）。

评估框架：2 测试集 × 2 模型 = 4 组交叉评估。

| 模型 | 测试集 | SBP MAE↓ | SBP STD↓ | DBP MAE↓ | DBP STD↓ |
|------|--------|:---:|:---:|:---:|:---:|
| base | 公开 (1582) | 3.19 | 5.11 | 1.68 | 2.86 |
| base | 自建 (273) | 9.17 | 10.93 | 17.82 | 8.12 |
| finetuned | 公开 (1582) | 3.13 | 5.07 | 1.64 | 2.78 |
| **finetuned** | **自建 (273)** | **3.89** | **6.05** | **2.21** | **3.45** |

**分析**：

- **源域内性能**（base → 公开测试）：SBP MAE=3.19, DBP MAE=1.68。7 特征 GPR 在源域上表现良好。与 1D-ResNet base（SBP 2.42）相比略高，但 GPR 额外提供了预测不确定性。
- **直接跨域性能**（base → 自建测试）：SBP MAE=9.17, DBP MAE=**17.82**。DBP 的跨域退化极为严重（从 1.68 → 17.82），远超 1D-ResNet 的对应退化（1.30 → 3.85）。这表明手工特征在 DBP 的跨域不变性不如端到端学习——7 个特征中的面积/时域特征可能过度拟合了源域的 DBP 分布。
- **微调后跨域性能**（finetuned → 自建测试）：**SBP 从 9.17 → 3.89（↓57.6%），DBP 从 17.82 → 2.21（↓87.6%）**。两者均满足 AAMI MAE≤5 标准。GPR 的联合训练策略在性能恢复幅度上非常显著。
- **源域性能保留**（finetuned → 公开测试）：SBP MAE=3.19→3.13, DBP MAE=1.68→1.64——**几乎没有退化**。这与 1D-ResNet 的微调形成鲜明对比：GPR 的联合训练（源域+目标域特征拼接为 5519×7 矩阵后再拟合一个 GP）天然地将两个域的数据平等对待，不像 DL 微调那样在梯度更新中存在"旧知识被覆盖"的问题。
- **与 1D-ResNet 的跨域性能对比**：

| 指标 | 1D-ResNet finetuned | GPR finetuned | 更优 |
|------|:---:|:---:|:---:|
| SBP MAE (自建) | 7.57 | **3.89** | GPR ✅ |
| SBP STD (自建) | 7.67 | **6.05** | GPR ✅ |
| DBP MAE (自建) | **2.08** | 2.21 | ≈ 持平 |
| DBP STD (自建) | **2.50** | 3.45 | 1D-ResNet ✅ |
| 源域性能保持 | ❌ 退化 | ✅ 稳定 | GPR ✅ |

GPR 在 SBP 预测和源域性能保持方面优势明显，1D-ResNet 在 DBP 预测稳定性上稍占优。两条路线在跨域迁移的不同维度上形成了互补。

### 5.4 域适应效果汇总

下表汇总了三条技术路线在自建测试集上的跨域预测性能：

| 路线 | 方法 | 目标域标注 | SBP MAE↓ | SBP STD↓ | DBP MAE↓ | DBP STD↓ |
|------|------|:---:|:---:|:---:|:---:|:---:|
| DL (v2.0) | 1D-ResNet base（无适应） | 0 | 12.87 | 11.61 | 3.85 | 4.13 |
| DL (v2.0) | 1D-ResNet 蒸馏微调 | 253 | 6.92 | 7.81 | 2.27 | 2.53 |
| GPR (v2.0) | GPR base（无适应） | 0 | 9.17 | 10.93 | 17.82 | 8.12 |
| GPR (v2.0) | GPR 联合训练微调 | 774 | 3.89 | 6.05 | 2.21 | 3.45 |
| DA (v3.0) | Feature DA 64-shot | 64 | 6.21 | 7.31 | 2.30 | 2.62 |
| DA (v3.0) | LwF 64-shot | 64 | 8.02 | 9.07 | 3.31 | 3.07 |

> 注：GPR 的目标域标注数为 774（78 维特征样本），DL 为 253（原始 PPG 样本），DA 为 64。三者使用不同的数据模态和样本量，直接对比需审慎。核心关注各路线自身 base→微调的改善幅度。
